# Student Performance Analysis

## Overview
This project analyzes student exam performance using Python, Pandas, Matplotlib, ML and Seaborn.

The objective is to understand how factors such as study hours, sleep hours, attendance percentage, and previous scores influence exam performance.

## Dataset Features
- Student ID
- Hours Studied
- Sleep Hours
- Attendance Percentage
- Previous Scores
- Exam Score

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Machine Learning
- train_test_split
- LinearRegression
  
## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Data Visualization
- Performance Trend Analysis
